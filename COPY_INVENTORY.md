# Реєстр копірайту · Moment Planner

Повний реєстр видимого користувачем тексту, згрупований за екранами. Інвентаризація — нічого не переписано.

Формат: **поточний текст** → `файл:рядок` → контекст (де/коли видно).

Плейсхолдери у `{фігурних дужках}` — динамічно згенеровані значення (дати, лічильники, назви міст через `Intl`/хелпери), не літерали.
